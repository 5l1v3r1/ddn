## 轻松上手

手动创建一个新链

```bash
// 安装 @ddn/ddn 命令行工具
$ yarn global add @ddn/ddn

// 创建区块链
$ ddn g blockchain MyBlockchain

// 启动
$ cd MyBlockchain
$ ./ddnd start
```
